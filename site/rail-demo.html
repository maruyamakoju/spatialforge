Technology

技術仕様

Depth Anything V2 Large + YOLOv8 欠陥検知を基盤とした鉄道専用の軌道点検エンジン。11種類の欠陥を自動分類・重要度判定。

📷

ハードウェア要件

既存の営業車両に搭載するだけ。特殊センサー不要。

  • 前方カメラ（720p以上）
  • LiDAR・レーザー不要
  • ステレオカメラ不要（単眼でOK）
  • Wi-Fi／LTE経由でクラウド送信
  • エッジ処理オプション対応
🧠

AI検知能力（11クラス分類）

Depth Anything V2 + YOLOv8 による2段階マルチレイヤー解析。

  • レールき裂・摩耗・波状摩耗・剥離
  • 締結装置の欠損・破損
  • まくらぎき裂・腐食
  • バラスト異状・継目異状・軌間異常
  • 4段階重要度自動判定（緊急/重要/軽微/参考）
📊

出力データ（実装済）

保線担当者がすぐに使える形式で自動レポート生成。

  • 欠陥分類レポート（HTML/PDF・日本語）
  • 深度カラーマップ + 欠陥バウンディングボックス
  • 重要度別サマリー・KM ポスト位置情報
  • 3D点群データ（.ply / .pcd）
  • REST API / JSON / 点検レポート一括出力
