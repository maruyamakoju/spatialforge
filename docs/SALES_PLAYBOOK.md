# RailScan 営業プレイブック
最終更新: 2026-03-01

## 戦略サマリー

**売るもの**: 鉄道軌道点検の自動化（PoC 3ヶ月 → 年間契約）
**売る相手**: 中小私鉄・地方第三セクター・地下鉄事業者の「施設部 保線課」

## アプローチ戦略

### フェーズ1: 初回コンタクト（Week 1-2）

**手段 1: LinkedInメッセージ**（最も効果的）
- 検索キーワード: `鉄道 保線 施設部長` / `railway maintenance Japan`
- 対象職種: 施設部長、保線課長、技術担当部長
- メッセージは下記テンプレート参照

**手段 2: 会社問い合わせフォーム経由のメール**
- 各社の「お問い合わせ」ページから施設・設備部門宛てに送信
- 件名を `AI軌道点検システム PoC の無償提供について` とする

**手段 3: 業界イベント**
- **鉄道技術展** (幕張メッセ、毎年11月) — ブース出展 or 名刺交換
- **土木学会 鉄道技術シンポジウム**
- **鉄道・電気技術者協会** のセミナー

### フェーズ2: 関係構築（Week 3-4）

- ミーティング設定 (Zoom or 訪問)
- 先方の路線の写真/動画を事前に収集し、サンプル解析結果を準備
- `proposal.html` のリンクを送付
- PoC 無償提供を提案（3ヶ月、1路線）

### フェーズ3: PoC → 契約（Month 2-3）

- PoC期間中に週次レポートを提供
- ROI算出（点検コスト削減額を定量化）
- 年間契約の見積もり提出

```

施設部 保線課 ご担当者様

突然のご連絡を失礼いたします。
AI軌道点検システム「RailScan」を開発・提供しているSpatialForgeと申します。

弊社の RailScan は、貴社の既存車載カメラ映像をリアルタイム AI 解析し、
建築限界違反・軌道障害物を自動検知するシステムです。

■ 主な特長
- 既存カメラに接続するだけ（ハードウェア追加不要）
- 114ms/フレームの高速推論（RTX GPU）
- 保線点検工数を最大47%削減（試算）
- 自動 PDF 点検レポート生成

実際のJR路線走行映像でのデモ結果はこちらでご確認いただけます：
https://maruyamakoju.github.io/spatialforge/rail-demo.html

現在、初期ユーザー様向けに **3ヶ月間の無償PoC** を提供しております。
貴社の路線映像（既存の前方カメラ映像で十分です）を用いた
カスタム解析デモをご用意できますので、
ぜひ一度オンラインでのご説明の機会をいただけないでしょうか。

ご都合のよいお日にちをいただけますと幸いです。

敬具

SpatialForge
founder@spatialforge.dev
https://maruyamakoju.github.io/spatialforge/railscan.html
```

## キーメッセージ（営業トーク）

### 「なぜ今か」
- 保線技術者の高齢化・採用難が深刻化（2030年までに保線員30%不足予測）
- 国交省の鉄道インフラDX推進方針（2025年度〜）
- 補助金活用可能（地方交通整備補助、DX関連助成）

### 「なぜRailScanか」
- 既存カメラでOK（ゼロ追加投資）
- 論文発表済みの信頼性（arXiv論文あり）
- 日本の保線規程に対応した分類体系

### 「リスクゼロの入口」
- 無償PoC 3ヶ月 → 効果確認後に契約判断
- 最小構成: 1路線 × 1区間から開始可
